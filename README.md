# Nuxt 2 Starter
A frontend starter with nuxt2 and essential plugins.

## prerequis
- node 18

## Plugins installed
- Webpack (builder)
- Vuetify 2 (css)
- Material design icons (icons)
- i18n (internationalization)
- VueX (Store)
- Axios (HTTP client)

## Default components
- App Bar
- Open-close App Menu
- Router
- Dark theme
- Reusable Snack that works with VueX
